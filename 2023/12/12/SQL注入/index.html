SQL 注入

条件： 用户参数可控

本质： 将用户输入的数据带入到数据库中当作sql代码执行

注释

GET 注入，在URL栏中显示

  • # –> %23
  • +的URL编码是空格

union 字段数要相同

  • 判断字段数 //order by 1

  • database() //显示当前所在数据库

information_schema //库
tables //存放了所有数据库中的表信息
columns //存放了所有数据库中的字段信息
. –> 表示 属于的关系
information_schema.tables information_schema中的tables表

table_schema 库名
table_name 表名
column_name 字段名

//我们要查询flag数据库中的表

group_concat() 连接

table_name=’flag’

select table_name from information_schema.tables where table_schema=’webug’;

select column_name from information_schema.column where table_name=’flag’;

select group_concat(flag) from flag;
