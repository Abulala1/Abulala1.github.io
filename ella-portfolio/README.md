# Ella — Personal Portfolio

A personal portfolio built with plain **HTML + CSS + vanilla JavaScript** —
no frameworks, no build step, 100% free to host.

纯 HTML + CSS + 原生 JavaScript 的个人网站，无需框架、无需打包工具，可完全免费部署。

## Files 文件结构

```
ella-portfolio/
├── index.html      → Home 首页（大字名 + 可拖动标签）
├── projects.html   → Projects 项目页
├── about.html      → About 关于页
├── contact.html    → Contact 联系页
├── style.css       → All styling 全部样式
├── main.js         → Menu / drag / scroll effects 交互脚本
└── assets/         → Your images 图片放这里
```

## Run locally 本地运行

1. Open this folder in VS Code（用 VS Code 打开这个文件夹）
2. Install the **Live Server** extension（安装 Live Server 扩展）
3. Right-click `index.html` → **Open with Live Server**
4. The site opens at `http://127.0.0.1:5500`

Or simply double-click `index.html` — it works without a server too.
（直接双击 index.html 也能打开。）

## Deploy free on GitHub Pages 免费部署

1. Create a public repo on GitHub named `<your-username>.github.io`
2. Push these files to the repo (root folder, not a subfolder)
3. Repo → Settings → Pages → Source: `main` branch → Save
4. Your site is live at `https://<your-username>.github.io` in ~1 minute

## Customize 自定义

- Search for `PLACEHOLDER` comments — replace fake info with yours
- Replace `hello@example.com` and `your-handle` links
- Put your photo in `assets/` and update `about.html`
